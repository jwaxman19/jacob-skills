# jacob-skills

A collection of practical agent skills for real work.

**Repo:** https://github.com/jwaxman19/jacob-skills

## Skills

- **[google-tac-tier2-casa-audit-prep](skills/google-tac-tier2-casa-audit-prep/)**  
  Review a codebase for Google TAC / CASA-style security and compliance prep.  
  Focus areas include OAuth flows, Gmail data handling, tokens, webhooks, APIs, and evidence gathering for questionnaires and audits.

- **[imessage-screenshot-generator](skills/imessage-screenshot-generator/)**  
  Generate realistic iPhone Messages (iMessage) screenshots as retina PNGs from a JSON conversation config—useful for mockups, stories, and presentations when you do not have a device. Requires Node.js and a local Chrome/Chromium.

## Install

Install from GitHub:

```bash
npx skills add jwaxman19/jacob-skills
```

## Contributing

From the repo root, run `./scripts/validate-skills.sh` before committing changes to skills.
